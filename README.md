# QuadB Tech Assignment
It is a web app which shows the series data in a beautiful UI. Complete web app has mobile/desktop friendly responsive UI.

# Technologies Used:
* ReactJS
* Tailwind CSS (For UI, UX)
* Netlify (for deployment)
